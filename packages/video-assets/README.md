# Storm Software - Video Assets

A project for generating various video and image assets used by Storm Software.

## Building

Run `nx build video-assets` to build the library.

## Running Development Server

Run `nx build video-assets` to build the library.

## Running unit tests

Run `nx test video-assets` to execute the unit tests via [Jest](https://jestjs.io).
